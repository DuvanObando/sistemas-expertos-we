# Sistemas Expertos Web

## Plataforma de Inteligencia Artificial — Expertos Inteligentes S.A.
https://sistemas-expertos-web-ing-duvanobando.netlify.app/

Plataforma web interactiva de sistemas expertos que implementa motores de inferencia basados en reglas de producción con encadenamiento hacia adelante. Cada módulo proporciona recomendaciones personalizadas y explicables en diferentes dominios de aplicación.

## Tecnologías Utilizadas

- **HTML5** - Estructura semántica y accesibilidad
- **CSS3** - Diseño responsive y moderno con animaciones
- **JavaScript** - Motores de inferencia y lógica de sistemas expertos

## Módulos Desarrollados

- 😊 **Bienestar Emocional** - Evaluación de salud mental y recomendaciones emocionales
- 📚 **Selección de Cursos** - Recomendaciones académicas personalizadas
- ⏰ **Gestión del Tiempo** - Optimización de productividad y administración del tiempo
- 💰 **Inversión** - Asesoramiento financiero según perfil de riesgo
- 🛒 **Compras Inteligentes** - Estrategias de compra informadas
- 📱 **Redes Sociales** - Optimización de contenido y estrategia digital
- ✈️ **Turismo** - Recomendaciones de destinos de viaje personalizados
- 🏃 **Alimentación Deportiva** - Planes nutricionales adaptativos
- 🧠 **Diagnóstico Psicológico** - Evaluación emocional orientativa (con advertencias éticas)
- 🎓 **Educativo Adaptativo** - Estrategias de aprendizaje personalizadas

## Cómo Ejecutar el Proyecto

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/[tu-usuario]/sistemas-expertos-web.git
   cd sistemas-expertos-web
   ```

2. **Abrir el proyecto:**
   - Abre el archivo `index.html` en tu navegador web preferido
   - O usa un servidor local como Live Server en VS Code

3. **Navegar por los módulos:**
   - Desde la página principal, selecciona cualquier módulo
   - Completa los formularios y recibe recomendaciones personalizadas
